Class Pitch
In: lib/pitch.rb
Parent: Object

This class represents a single pitch in an MLB baseball game. Most of the attributes represent the physics of the pitch thrown.

Pitch Type Codes 

  FS = Splitter
  SL = Slider
  FF = Fastball
  SI = Sinker
  CH = Change
  FA = Fastball
  CU = Curve
  FC = Cutter
  KN = Knuckle
  KC = Knuckle Curve
